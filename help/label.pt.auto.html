Ativar rastreamento do Google Analytics?
Quando esse recurso está ativo, todas as páginas HTML servidas por este site terão o Javascript adicionado automaticamente para ativar o rastreamento através de serviços de análise, como Google Analytics, MyBlogLog e Quantcast.

No entanto, como cada site deve estar associado a uma conta, você precisará usar a página Contas do Analytics no menu Logs e relatórios para inserir os IDs de conta para cada serviço antes que o rastreamento seja ativado. Para obter os IDs da conta, primeiro você deve se inscrever em um desses serviços de rastreamento, o que pode ser feito nos sites deles.
