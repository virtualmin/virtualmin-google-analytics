ID trang web Piwik
Nếu bạn đã cài đặt gói phân tích Piwik miễn phí (sử dụng Virtualmin hoặc thủ công), mỗi trang web mà nó báo cáo được xác định bởi một ID trang web số. Đây thường là một số nhỏ (như 5) và có thể thấy trong Javascript mà Piwik gợi ý bạn thêm vào các trang web của mình trong biến piwik_idsite .
