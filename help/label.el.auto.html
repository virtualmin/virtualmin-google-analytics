Ενεργοποίηση παρακολούθησης αναλυτικών στοιχείων;
Όταν αυτή η λειτουργία είναι ενεργή, όλες οι σελίδες HTML που θα εξυπηρετούνται από αυτόν τον ιστότοπο θα έχουν αυτόματα προσθήκη Javascript για να επιτρέπουν την παρακολούθηση μέσω υπηρεσιών ανάλυσης όπως το Google Analytics, το MyBlogLog και το Quantcast.

Ωστόσο, επειδή κάθε ιστότοπος πρέπει να συσχετιστεί με έναν λογαριασμό, θα πρέπει να χρησιμοποιήσετε τη σελίδα Λογαριασμοί Analytics κάτω από το μενού Logs and Reports για να εισαγάγετε αναγνωριστικά λογαριασμού για κάθε υπηρεσία προτού ενεργοποιηθεί η παρακολούθηση. Για να αποκτήσετε αναγνωριστικά λογαριασμού, πρέπει πρώτα να εγγραφείτε σε μία από αυτές τις υπηρεσίες παρακολούθησης, οι οποίες μπορούν να πραγματοποιηθούν στους ιστότοπούς τους.
