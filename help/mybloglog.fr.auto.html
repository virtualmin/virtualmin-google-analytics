ID de compte MyBlogLog
Pour activer le suivi MyBlogLog pour ce serveur virtuel, vous devez d'abord vous inscrire à un compte sur http://www.mybloglog.com/ . Une fois cela fait, vous recevrez Javascript à ajouter à votre site Web, qui contient un ID comme 2006033112252119 . Au lieu d'ajouter ce Javascript manuellement, entrez simplement l'ID dans ce champ.
