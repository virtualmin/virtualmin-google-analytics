Povolit sledování analytiků?
Když je tato funkce aktivní, budou všechny stránky HTML poskytované tímto webem automaticky přidávány do Javascriptu, aby bylo možné sledovat pomocí analytických služeb, jako jsou Google Analytics, MyBlogLog a Quantcast.

Protože však každý web musí být spojen s účtem, budete muset před aktivací sledování pomocí stránky Účty Analytics v nabídce Protokoly a přehledy zadat ID účtu pro každou službu. Chcete-li získat ID účtu, musíte se nejprve zaregistrovat pro jednu z více těchto sledovacích služeb, které lze provést na jejich webových stránkách.
