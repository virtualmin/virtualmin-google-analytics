MyBlogLog account ID
Om een MyBlogLog tracering aan te zetten voor deze virtuele server, moet u eerst een account aanmaken bij http://www.mybloglog.com/. Als u dit heeft gedaan krijgt u een Javascript om in uw website te zetten, dit script bevat een ID, bijvoorbeeld 2006033112252119. In plaats van dit script handmatig in uw website te zetten kan dit ook automatisch gedaan worden voor u door het ID in dit veld in te vullen.
