MyBlogLog-konto-ID
För att aktivera MyBlogLog-spårning för den virtuella servern måste du först registrera dig för ett konto på http://www.mybloglog.com/ . När detta är gjort får du Javascript att lägga till på din webbplats, som innehåller ett ID som 2006033112252119 . Istället för att lägga till detta Javascript manuellt, ange bara ID: n i det här fältet.
