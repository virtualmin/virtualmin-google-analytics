URL pelayan analisis Piwik
Untuk menggunakan Piwik untuk menghasilkan statistik di laman web anda, Virtualmin perlu mengetahui URL asas pemasangan Piwik untuk menghantar statistik. Ini biasanya seperti http://yourdomain.com/piwik/ . Sekiranya anda memasang Piwik menggunakan Virtualmin, anda dapat melihat URL di halaman Pasang Skrip .
