Identifiant du site Piwik
Si vous avez installé le package d'analyse Piwik gratuit (à l'aide de Virtualmin ou manuellement), chaque site sur lequel il rend compte est identifié par un ID de site numérique. Il s'agit généralement d'un petit nombre (comme 5), et peut être vu dans le Javascript que Piwik vous suggère d'ajouter à vos pages Web dans la variable piwik_idsite .
