ID de compte Quantcast
Pour activer le suivi Quantcast pour ce serveur virtuel, vous devez d'abord ouvrir un compte sur http://www.quantcast.com/ . Une fois cela fait, vous recevrez Javascript à ajouter à votre site Web, qui contient un ID comme p-19VXA69bF5MtA . Au lieu d'ajouter ce Javascript manuellement, entrez simplement l'ID dans ce champ.
