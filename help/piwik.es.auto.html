ID del sitio de Piwik
Si ha instalado el paquete de análisis Piwik gratuito (ya sea utilizando Virtualmin o manualmente), cada sitio sobre el que informa se identifica mediante una ID numérica del sitio. Este es típicamente un número pequeño (como 5), y se puede ver en el Javascript que Piwik sugiere que agregue a sus páginas web en la variable piwik_idsite .
