ID sito Piwik
Se è stato installato il pacchetto di analisi Piwik gratuito (utilizzando Virtualmin o manualmente), ogni sito su cui viene segnalato viene identificato da un ID sito numerico. Questo è in genere un numero piccolo (come 5) e può essere visto nel Javascript che Piwik suggerisce di aggiungere alle tue pagine web nella variabile piwik_idsite .
