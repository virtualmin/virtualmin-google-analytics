Clicky account ID
Om een Clicky tracering aan te zetten voor deze virtuele server moet u eerst een account aanmaken op http://getclicky.com/. Zodra u dit gedaan heeft krijgt u een Javascript om in uw website te zetten, hierin zit een ID, bijvoorbeeld 12555. U hoeft dit niet handmatig in uw website te zetten, als u het ID in dit veld zet zal het automatisch in uw website geplaatst worden.
