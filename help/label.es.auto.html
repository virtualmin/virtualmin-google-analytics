¿Habilitar el seguimiento de Analytics?
Cuando esta función está activa, todas las páginas HTML que sirve este sitio web tendrán Javascript agregado automáticamente para permitir el seguimiento a través de servicios de análisis como Google Analytics, MyBlogLog y Quantcast.

Sin embargo, debido a que cada sitio debe estar asociado con una cuenta, deberá utilizar la página Cuentas de Analytics en el menú Registros e Informes para ingresar las ID de cuenta para cada servicio antes de activar el seguimiento. Para obtener ID de cuenta, primero debe registrarse en uno de estos servicios de seguimiento, que se puede hacer en sus sitios web.
