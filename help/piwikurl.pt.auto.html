URL do servidor de análise Piwik
Para usar o Piwik para gerar estatísticas em seu site, o Virtualmin precisa saber o URL base da instalação do Piwik para o qual enviar estatísticas. Isso geralmente é algo como http://yourdomain.com/piwik/ . Se você instalou o Piwik usando o Virtualmin, poderá ver a URL na página Instalar scripts .
