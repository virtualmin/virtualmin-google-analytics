Activer le suivi analytique?
Lorsque cette fonctionnalité est active, Javascript sera ajouté automatiquement à toutes les pages HTML desservies par ce site Web pour permettre le suivi via des services d'analyse tels que Google Analytics, MyBlogLog et Quantcast.

Cependant, étant donné que chaque site doit être associé à un compte, vous devrez utiliser la page Comptes Analytics dans le menu Journaux et rapports pour entrer les ID de compte pour chaque service avant que le suivi ne soit activé. Pour obtenir des identifiants de compte, vous devez d'abord vous inscrire à l'un de plusieurs de ces services de suivi, ce qui peut être fait sur leurs sites Web.
