Activați urmărirea analizelor?
Când această caracteristică este activă, toate paginile HTML servite de acest site vor avea adăugat automat Javascript pentru a permite urmărirea prin intermediul serviciilor de analiză precum Google Analytics, MyBlogLog și Quantcast

Cu toate acestea, deoarece fiecare site trebuie asociat cu un cont, va trebui să utilizați pagina Conturi Analytics din meniul Jurnaluri și rapoarte pentru a introduce ID-uri de cont pentru fiecare serviciu înainte de a fi activat. Pentru a obține un ID de cont, trebuie să vă înscrieți mai întâi la unul dintre mai multe dintre aceste servicii de urmărire, care pot fi efectuate pe site-urile lor web.
