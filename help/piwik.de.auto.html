Piwik Site ID
Wenn Sie das kostenlose Piwik-Analysepaket installiert haben (entweder mit Virtualmin oder manuell), wird jede Site, über die berichtet wird, durch eine numerische Site-ID identifiziert. Dies ist normalerweise eine kleine Zahl (wie 5) und kann im Javascript piwik_idsite werden, das Piwik vorschlägt, dass Sie Ihre Webseiten in der Variablen piwik_idsite .
