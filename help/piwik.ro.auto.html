ID site-ul Piwik
Dacă ați instalat pachetul gratuit de analiză Piwik (fie utilizând Virtualmin, fie manual), fiecare site pe care îl raportează este identificat printr-un ID numeric al site-ului. Acesta este de obicei un număr mic (cum ar fi 5) și poate fi văzut în Javascript că Piwik sugerează să adăugați la paginile dvs. web variabila piwik_idsite .
