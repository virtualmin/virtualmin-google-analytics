ID do site Piwik
Se você instalou o pacote de análise Piwik gratuito (usando o Virtualmin ou manualmente), cada site relatado é identificado por um ID numérico do site. Normalmente, esse é um número pequeno (como 5) e pode ser visto no Javascript que o Piwik sugere que você adicione às suas páginas da Web na variável piwik_idsite .
