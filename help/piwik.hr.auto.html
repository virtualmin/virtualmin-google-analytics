ID web lokacije Piwik
Ako ste instalirali besplatan paket analitike Piwik (bilo pomoću Virtualmin-a ili ručno), svaka web lokacija o kojoj izvještava identificira se numeričkim ID-om web mjesta. To je obično mali broj (poput 5), a može se vidjeti u Javascript-u koji Piwik predlaže da dodate na svoje web stranice u varijabli piwik_idsite .
