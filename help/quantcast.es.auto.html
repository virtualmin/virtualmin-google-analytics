Quantcast ID de cuenta
Para habilitar el seguimiento de Quantcast para este servidor virtual, primero debe registrarse para obtener una cuenta en http://www.quantcast.com/ . Una vez hecho esto, se le dará Javascript para agregar a su sitio web, que contiene una identificación como p-19VXA69bF5MtA . En lugar de agregar este Javascript manualmente, solo ingrese la ID en este campo.
