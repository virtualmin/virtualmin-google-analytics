URL сервера аналитики Piwik
Чтобы использовать Piwik для генерации статистики на вашем веб-сайте, Virtualmin должен знать базовый URL-адрес установки Piwik для отправки статистики. Обычно это что-то вроде http://yourdomain.com/piwik/ . Если вы установили Piwik с помощью Virtualmin, вы можете увидеть URL на странице Install Scripts .
