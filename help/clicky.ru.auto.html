Клик идентификатор аккаунта
Чтобы включить отслеживание кликов для этого виртуального сервера, сначала необходимо зарегистрировать учетную запись на http://getclicky.com/ . Как только это будет сделано, вам будет предоставлен Javascript для добавления на ваш сайт, который содержит идентификатор, например, 12555 . Вместо добавления этого Javascript вручную, просто введите идентификатор в это поле.
