ID laman web Piwik
Sekiranya anda telah memasang pakej analisa Piwik percuma (sama ada menggunakan Virtualmin atau secara manual), setiap laman web yang dilapor akan dikenal pasti oleh ID laman web berangka. Ini biasanya sebilangan kecil (seperti 5), dan dapat dilihat dalam Javascript yang dicadangkan oleh Piwik agar anda menambahkan ke laman web anda dalam pemboleh ubah laman web piwik_idsite .
