Dayakan penjejakan Analisis?
Apabila ciri ini aktif, semua halaman HTML yang dilayani oleh laman web ini akan mempunyai Javascript yang ditambahkan secara automatik untuk membolehkan penjejakan melalui perkhidmatan analisis seperti Google Analytics, MyBlogLog dan Quantcast.

Namun, kerana setiap laman web mesti dikaitkan dengan akaun, anda perlu menggunakan halaman Akaun Analitis di bawah menu Log dan Laporan untuk memasukkan ID akaun untuk setiap perkhidmatan sebelum penjejakan diaktifkan. Untuk mendapatkan ID akaun, anda mesti terlebih dahulu mendaftar ke salah satu daripada lebih banyak perkhidmatan penjejakan ini, yang boleh dilakukan di laman web mereka.
