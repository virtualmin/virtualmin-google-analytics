Идентификатор учетной записи MyBlogLog
Чтобы включить отслеживание MyBlogLog для этого виртуального сервера, сначала необходимо зарегистрировать учетную запись по адресу http://www.mybloglog.com/ . Как только это будет сделано, вам будет предоставлен Javascript для добавления на ваш сайт, который содержит идентификатор, например, 2006033112252119 . Вместо добавления этого Javascript вручную, просто введите идентификатор в это поле.
