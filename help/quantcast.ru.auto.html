Идентификатор аккаунта Quantcast
Чтобы включить отслеживание Quantcast для этого виртуального сервера, сначала необходимо зарегистрировать учетную запись на http://www.quantcast.com/ . Как только это будет сделано, вам будет предоставлен Javascript для добавления на ваш сайт, который содержит идентификатор, например, p-19VXA69bF5MtA . Вместо добавления этого Javascript вручную, просто введите идентификатор в это поле.
