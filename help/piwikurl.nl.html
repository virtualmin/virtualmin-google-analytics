Piwik analytische server URL
Om door Piwik statistieken te genereren op uw website heeft Virtualmin de basis URL nodig van de Piwik installatie waar de statistieken naar toe gestuurd moeten worden. Dit lijkt normaal gesproken op http://yourdomain.com/piwik/. Indien u Piwik geinstalleerd heeft via Virtualmin kunt u de URL zien op de Installeer Scripts pagina.
