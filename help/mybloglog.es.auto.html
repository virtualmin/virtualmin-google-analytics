ID de cuenta MyBlogLog
Para habilitar el seguimiento de MyBlogLog para este servidor virtual, primero debe registrarse para obtener una cuenta en http://www.mybloglog.com/ . Una vez hecho esto, se le dará Javascript para agregar a su sitio web, que contiene una identificación como 2006033112252119 . En lugar de agregar este Javascript manualmente, solo ingrese la ID en este campo.
