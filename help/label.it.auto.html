Abilita il monitoraggio di Analytics?
Quando questa funzione è attiva, tutte le pagine HTML servite da questo sito Web avranno Javascript aggiunto automaticamente per consentire il tracciamento tramite servizi di analisi come Google Analytics, MyBlogLog e Quantcast.

Tuttavia, poiché ogni sito deve essere associato a un account, è necessario utilizzare la pagina Account Analytics nel menu Registri e rapporti per inserire gli ID account per ciascun servizio prima di attivare il tracciamento. Per ottenere gli ID account, è necessario prima registrarsi a uno di questi servizi di tracciamento, che può essere eseguito sui loro siti Web.
