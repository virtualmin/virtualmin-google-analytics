URL du serveur d'analyse Piwik
Afin d'utiliser Piwik pour générer des statistiques sur votre site Web, Virtualmin doit connaître l'URL de base de l'installation Piwik à laquelle envoyer les statistiques. Il s'agit généralement de quelque chose comme http://yourdomain.com/piwik/ . Si vous avez installé Piwik à l'aide de Virtualmin, vous pouvez voir l'URL sur la page Installer les scripts .
